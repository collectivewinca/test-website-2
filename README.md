# Test Website 2

Test website based on the Tailwind Next.js starter blog stack.

## Purpose

Provide a sandbox for validating website and blog changes quickly.

## Expected Experience

- Rapid local preview
- Safe experimentation with content and UI
- Repeatable test deployments

## Getting Started

```bash
yarn install
yarn dev
```

## Notes

Use this as an experimental environment rather than a long-term production base.
